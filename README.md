# Sales Performance Dashboard

**Description:** Daily sales performance overview for a food chain with 27 branches across the Philippines. Tracks sales trends, branch performance, and key product insights.

---

## Tools Used
- Google Sheets
- Pivot Tables, Formulas
- Charts & Conditional Formatting

---

## Insights
- NCR and VisMin branches had the highest daily sales
- Top-selling products identified for stock optimization
- Daily trends help management make quick operational decisions
